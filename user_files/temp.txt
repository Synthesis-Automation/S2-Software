# Reaction Protocol Format Version 1.1
# No space in names, ';' as seperator, 'mL' for liquids, 'mmol' for solids, program ignores chemical name start with '$'.
# Example: 

KF (0.005 mmol); Test (0.1 mL); Reaction-temperature (25.0 degree); Reaction-time (6.0 h).
CsF (0.005 mmol); Test (0.2 mL); Reaction-temperature (25.0 degree); Reaction-time (6.0 h).
